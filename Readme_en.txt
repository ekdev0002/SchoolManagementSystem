Informations
-------------------------------------------------------------------------------
School management application written in java

---------------------------------features----------------------------------

1) Application management
2) Student monitoring
3) notes management
4) Planning of lessons and homework
5) printing of school report

-----------------------------------Instructions-----------------------------------
1) Install MySQL
2) Create a schoolsm database and import the data from the DB.sql file into it
3) Import the project under Eclipse and configure the parameters in the database
   in the configuration.xml file
