Information
-------------------------------------------------------------------------------
Application de gestion d'école écrite en java 

---------------------------------Fonctionnalités----------------------------------

1) Gestion des candidatures
2) Suivi des éléves
3) gestion des notes
4) Plannification des cours et des devoirs
5) impression de bulletins

-----------------------------------Instructions-----------------------------------

1) Installer MySQL
2) Créer une base de donnée schoolsm et importez y les données du fichier DB.sql
3) Importez le projet sous Eclipse et configurer les parametres à la base de  données
   dans le fichier configuration.xml

Codage
